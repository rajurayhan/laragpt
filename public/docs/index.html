Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

curl --request POST \
    "http://localhost:8000/api/completion" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"

);

const headers = {
    "Content-Type": "application/json",

Meeting Transcript

Creae New Meeting

Example request:
curl --request POST \
    "http://localhost:8000/api/project-summery" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"transcriptText\": \"quia\",
    \"projectName\": \"inventore\",
    \"clientPhone\": \"est\",
    \"clientEmail\": \"orval17@example.com\",
    \"clientWebsite\": \"nihil\"
}"

const url = new URL(
    "http://localhost:8000/api/project-summery"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "transcriptText": "quia",
    "projectName": "inventore",
    "clientPhone": "est",
    "clientEmail": "orval17@example.com",
    "clientWebsite": "nihil"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/project-summery

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

transcriptText   string   

The text of the transcript. Example: quia

projectName   string   

The name of the project. Example: inventore

clientPhone   string  optional  

The phone number of the client. Example: est

clientEmail   string  optional  

The email of the client. Example: orval17@example.com

clientWebsite   string  optional  

The website of the client. Example: nihil
